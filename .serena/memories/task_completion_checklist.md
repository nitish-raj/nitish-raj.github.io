# Task Completion Checklist

## Before Completing Any Task

### Build and Test
- [ ] Run `quarto render` to ensure site builds successfully
- [ ] Run `quarto preview` and test the site locally
- [ ] Check responsive design at different screen sizes
- [ ] Test dark/light mode toggle functionality
- [ ] Verify all internal and external links work
- [ ] Test interactive JavaScript features

### Code Quality
- [ ] Review code follows established style conventions
- [ ] Check for proper semantic HTML5 usage
- [ ] Verify accessibility features (ARIA labels, alt text)
- [ ] Ensure consistent naming conventions
- [ ] Check for proper error handling in JavaScript

### Content Review
- [ ] Proofread all text content for typos
- [ ] Verify YAML frontmatter is correct
- [ ] Check image alt text is descriptive
- [ ] Ensure consistent formatting across pages
- [ ] Validate any embedded code or examples

### Performance and SEO
- [ ] Optimize image sizes and formats
- [ ] Check page load speed
- [ ] Verify meta tags and SEO elements
- [ ] Test site on different browsers
- [ ] Check mobile performance

### Final Checks
- [ ] Run `git status` to review changes
- [ ] Ensure no sensitive information is committed
- [ ] Verify all files are properly saved
- [ ] Check that the site builds without warnings
- [ ] Test the deployed version if applicable
