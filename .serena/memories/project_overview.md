# Project Overview

## Purpose
This is Nitish Raj's personal website/portfolio built with Quarto. It showcases his professional background as a Senior Data Engineer specializing in Data Engineering and MLOps.

## Tech Stack
- **Static Site Generator**: Quarto
- **Styling**: SCSS with Bootstrap themes (flatly/darkly)
- **Fonts**: Google Fonts (Lato, Roboto) + Font Awesome icons
- **JavaScript**: Vanilla ES6+ for interactive features
- **Deployment**: GitHub Pages (docs/ output directory)
- **Domain**: rajnitish.com

## Project Structure
- `_quarto.yml` - Main Quarto configuration
- `_format.yml` - HTML format configuration with themes and styling
- `_website.yml` - Website-specific metadata
- `styles.scss` - Custom SCSS styles with Bootstrap variables
- `script/script.js` - Interactive JavaScript features
- `images/` - Static assets (logos, profile images)
- `_partials/` - Reusable HTML components
- `*.qmd` - Quarto markdown files for pages
- `docs/` - Built site output directory

## Key Features
- Responsive design with mobile-first approach
- Dark/light mode toggle
- Smooth scrolling and interactive elements
- Skill badges with hover effects
- Experience section with expandable details
- Social media integration
- SEO optimization with proper metadata
